Property Manager

A website that displays properties for rent/sale, and has an admin area for tracking rents/maintenance on rentals.

Front Page

As a user, I can see featured properties from several categories.

As a user, I can click a link to see property details.

As a user, I can click a link to filter properties by category

As a user, I can search for properties by size, price, neighborhood, etc.

As a renter, I can login to make an online rent payment or request maintenance

As a property manager, I can login to manage website/business details
